# StudyFlow - AI-Powered Study Planner

Convert unstructured syllabus PDFs into structured answer key books and personalized study schedules with intelligent AI automation.

## Features

- **AI Syllabus Analysis**: Automatic extraction of units, topics, and subtopics from PDF documents
- **Answer Key Generation**: Professional book-quality study materials with exam-oriented content
- **Smart Scheduling**: Time-bound study plans with personalized calendar integration
- **Calendar Sync**: Google Calendar API integration with ICS export functionality
- **Responsive Design**: Access your study materials on any device

## Technology Stack

### Backend
- Python
- FastAPI

### AI Processing
- Large Language Models
- PyPDF2 for PDF extraction

### Frontend
- HTML5 & CSS3
- Tailwind CSS
- Print-ready layout

### Authentication & Database
- Supabase Authentication
- PostgreSQL

### Calendar Integration
- Google Calendar API
- OAuth 2.0
- ICS file generation

## Getting Started

1. Open `new.html` in your web browser
2. Upload your syllabus PDF
3. Let AI analyze and generate study materials
4. Export to calendar or download as PDF

## Workflow

1. **Authentication** - Secure login via Supabase
2. **Upload** - Upload syllabus PDF documents
3. **AI Analysis** - Extract units, topics & subtopics
4. **Content Generation** - Generate answer key books in HTML
5. **Scheduling** - Create time-bound study plans
6. **Sync** - Export ICS & Google Calendar synchronization
